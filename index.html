SRR v5 알고리즘 설명

SRR이란?
Sleeping Respiratory Rate — 수면 호흡수. 가슴/배가 한 번 올라갔다 내려오면 = 1회 호흡으로 셉니다.

1. 다중 채널 분석
R/G/B 3개 채널을 독립 추적합니다. 프레임 밝기(조도)에 따라 최적 채널을 자동 선택합니다. 어두울 때 R채널, 보통 조명에서 G채널이 유리합니다.

2. 적응형 밴드패스 필터
고역통과(0.1Hz) + 저역통과(1.5Hz) 캐스케이드 필터로 호흡 신호 대역(분당 6~90회)만 정밀 추출합니다.

3. 3중 교차 검증
자기상관 + FFT 주파수 분석 + 피크 카운팅 — 3가지 방법의 결과가 일치하면 신뢰도가 올라가고, 가중 평균으로 최종 BPM을 결정합니다.

4. 칼만 필터
BPM 결과를 칼만 필터로 시계열 추적하여 순간적인 튐 없이 안정적인 수치를 표시합니다.

5. 모션 게이트 (4단계)
자이로+가속도계 상보 필터 융합 → 광학 흐름 → 밝기 변화율 감지. 카메라 흔들림 시 해당 프레임을 자동 제외합니다.

6. Web Worker
무거운 분석 연산을 별도 스레드에서 처리하여 화면 끊김을 방지합니다.
