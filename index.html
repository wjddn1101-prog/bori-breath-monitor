SRR 알고리즘 설명

SRR이란?
Sleeping Respiratory Rate — 수면 호흡수. 가슴/배가 한 번 올라갔다 내려오면 = 1회 호흡으로 셉니다.

1. ROI 밝기 추적
선택한 영역의 평균 밝기를 매 프레임 기록합니다. 가슴이 오르면 빛 반사가 변하여 밝기 신호가 주기적으로 변합니다.

2. 신호 전처리
선형 추세 제거(조명 변화 보정) → 이동평균 스무딩(노이즈 제거) → 정규화

3. 자기상관 분석
신호가 자기 자신과 얼마나 반복되는지 계산합니다. 호흡이 규칙적이면 호흡 주기에 해당하는 지점에서 강한 피크가 나타납니다.

4. 피크 카운팅 교차 검증
자기상관 결과를 시간축 피크 검출로 이중 확인하여 정확도를 높입니다.

5. 모션 게이트
카메라 흔들림이나 강아지 움직임 감지 시 해당 프레임을 자동 제외합니다.

촬영 팁: 삼각대나 받침대에 폰 고정 / 일정한 조명 / 보리가 깊이 잠든 상태에서 측정하세요.
