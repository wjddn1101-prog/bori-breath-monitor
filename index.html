SRR v6 + 타일 융합 ML 알고리즘 설명

SRR이란?
Sleeping Respiratory Rate — 수면 호흡수. 가슴/배가 한 번 올라갔다 내려오면 = 1회 호흡으로 셉니다.

1. ROI 자동 탐지
측정 시작 후 5초간 화면을 3×3 그리드(9개 영역)로 나눠 각 영역의 호흡 대역 에너지를 비교합니다. 에너지가 가장 안정적인 영역(보리 가슴/배)을 자동으로 측정 영역으로 선택합니다.

2. 다중 타일 분석
선택된 ROI를 다시 4×3 타일로 나눠 각 타일의 신호를 독립 추적합니다. 한 점만 튀는 노이즈 대신 여러 타일이 함께 움직이는 패턴을 우선적으로 사용합니다.

3. 밝기 + 로컬 모션 융합
각 타일에서 R/G/B 밝기 변화와 세로 방향 로컬 모션을 함께 계산합니다. 저조도에서는 밝기만 믿지 않고, 털 결이나 미세한 움직임 패턴까지 함께 반영해 더 안정적으로 호흡 신호를 만듭니다.

4. coherence 기반 타일 선택
최근 2~3초 동안 서로 위상이 맞는 타일들만 높은 가중치를 줍니다. 여러 타일이 비슷한 주기로 움직일수록 신뢰도가 올라가고, 한 타일만 튀는 경우는 억제합니다.

5. 적응형 밴드패스 + 3중 교차 검증
고역통과(0.1Hz) + 저역통과(1.3Hz) 필터로 호흡 대역만 추출한 뒤, 자기상관 + FFT + 피크 카운팅 3가지 방법이 서로 맞는지 교차 검증합니다.

6. IMU 흔들림 게이트 + TF.js 품질 분류기
자이로/가속도계로 폰 흔들림을 감지해 강한 흔들림 구간은 버립니다. 이후 TensorFlow.js 분류기가 남은 신호가 실제 호흡인지 마지막으로 한 번 더 걸러냅니다.

7. 적응 학습 + 칼만 필터 + Web Worker
결과 피드백에서 성공한 조도·감도 환경만 학습해 다음 측정에 반영합니다. 최종 BPM은 칼만 필터로 안정화하고, 무거운 연산은 Web Worker에서 처리해 화면 끊김을 줄입니다.

저조도 팁: 폰은 최대한 고정 / 가슴·배가 넓게 보이도록 ROI 확보 / 너무 어두우면 플래시보다 약한 보조 조명 권장.
처음 3~5회는 측정 후 피드백을 눌러주세요 — 보리 맞춤 모드가 더 빨리 안정화됩니다.
